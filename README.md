# WeatherNow

Aplicação web para consulta de clima em tempo real utilizando JavaScript e integração com a API OpenWeather.

O projeto permite pesquisar cidades e visualizar informações climáticas atualizadas, como temperatura, sensação térmica, umidade, velocidade do vento e condições do clima.

# 🚀 Funcionalidades

- Consulta de clima em tempo real
- Pesquisa dinâmica de cidades
- Exibição de:
  - Temperatura atual
  - Temperatura máxima e mínima
  - Sensação térmica
  - Umidade
  - Velocidade do vento
  - Condição climática
- Integração com ícones dinâmicos da OpenWeather
- Interface responsiva e moderna
